*Depression i.e. any antidepressant*
*GEE analysis


use "final_analysis_file.dta", clear

*replace "missing" categories as "." where not being included as dummy 
replace deprivation=. if deprivation==6

replace matage_cat=. if matage_cat==5

replace parity_cat=. if parity_cat==3

replace apgarcat=. if apgarcat==4 

replace bw_cent=. if bw_cent==8

*create varlist
vl create SENlist = (anySEN ASD sens_impair com_prob learning_dif physical_med_dif emotional_dif)

*GEE models looking at outcome of anySEN and individual types of SEN
*clear estimates stored
eststo clear
estimates clear 

sort ALF_PE YEAR
xtset ALF_PE YEAR

*Run qic statistic for different correlation structures

/*qic anySEN Depression_bcd, i(ALF_PE) t(YEAR) family(binomial) link(logit) corr(independent) eform
qic anySEN Depression_bcd, i(ALF_PE) t(YEAR) family(binomial) link(logit) corr(exchangeable) eform
qic anySEN Depression_bcd, i(ALF_PE) t(YEAR) family(binomial) link(logit) corr(unstructured) eform*/
*Independent structure remains lowest qic

*Run loops for each exposure category e.g. Depression (wide definition) compared to comparison group 
**-------
** BCDvA (Depression - wide definition)
**-------
*univariate
foreach v of varlist $SENlist {
	xi:xtgee i.`v' Depression_bcd, i(ALF_PE) t(YEAR) family(binomial) link(logit) corr(independent) eform vce(robust)
eststo `v'_uni
}

*multivariate1 - sex, deprivation, ethnicity, age
*set base to 5 for deprivation; female and white automatically coded
foreach v of varlist $SENlist {
	xi:xtgee i.`v' i.gender_cat b5.deprivation i.ethnicity_cat AGE_PUPILCENSUS Depression_bcd, i(ALF_PE) t(YEAR) family(binomial) link(logit) corr(independent) eform vce(robust)
eststo `v'_mv1
}

*multivariate2 - sex, deprivation, ethnicity, age, maternal age, maternal smoking, parity, multiple birth
*set base to 5 for deprivation, 2 for mat age category; female, white, non smoker, nulliparous, multiple=0 automatically coded
foreach v of varlist $SENlist {
xi:xtgee i.`v' i.gender_cat b5.deprivation i.ethnicity_cat AGE_PUPILCENSUS b2.matage_cat b0.mat_smoking_cat i.parity_cat i.multiple_binary Depression_bcd, i(ALF_PE) family(binomial) link(logit) corr(independent) eform vce(robust) 
eststo `v'_mv2
}

*multivariate3 - sex, deprivation, ethnicity, maternal age, maternal smoking, parity, multiple birth, epilepsy
*set base to 5 for deprivation, 2 for mat age category; female, white, non smoker, nulliparous, multiple=0, no epilepsy automatically coded
foreach v of varlist $SENlist {
xi:xtgee i.`v' i.gender_cat b5.deprivation i.ethnicity_cat AGE_PUPILCENSUS b2.matage_cat b0.mat_smoking_cat i.parity_cat i.multiple_binary i.epilepsy Depression_bcd, i(ALF_PE) family(binomial) link(logit) corr(independent) eform vce(robust) 
eststo `v'_mv3
}

*multivariate4 - mediators added: mode of delivery, birth weight centile, gestational age, apgar
*set base to 5 for deprivation, 2 for mat age category, 3 for apgar, 4 for bw_cent; female, white, non smoker, nulliparous, SVD, multiple=0, no epilepsy automatically coded
foreach v of varlist $SENlist {
xi:xtgee i.`v' i.gender_cat b5.deprivation i.ethnicity_cat AGE_PUPILCENSUS b2.matage_cat b0.mat_smoking_cat i.parity_cat i.multiple_binary i.epilepsy b4.bw_cent GEST_AGE b3.apgarcat i.delivery_cat Depression_bcd, i(ALF_PE) family(binomial) link(logit) corr(independent) eform vce(robust)  
eststo `v'_mv4
}

*create table for all estimates stored
*Esttab
esttab _all using "results.csv", cells ("b(label(OR)) ci_l(label(Lower limit)) ci_u (label(Upper limit)) p(label(p-value))") eform ///
title("Gee analysis results- Depression BCDvA") nolabel plain noobs ///
stats(N,fmt("%9.0fc"))
replace

*repeat for other exposure groups e.g. treated depression


log close
